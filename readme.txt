Este bot incluye:

Comandos: El bot tiene una lista de comandos disponibles que los usuarios pueden utilizar para interactuar con él. Los comandos incluyen cmds, hello, bye, random_password, random_emoji, flip_coin, joined_server, joined_discord, help_environment y test.
Ayuda al medio ambiente: El comando help_environment proporciona información y consejos para cuidar el medio ambiente. Los usuarios pueden seleccionar una categoría (reducir cantidad de residuos, opciones de reciclaje o recomendaciones) y el bot mostrará información relevante.
Generación de contraseñas aleatorias: El comando random_password genera una contraseña aleatoria de una longitud determinada por el usuario.
Generación de emojis aleatorios: El comando random_emoji genera un emoji aleatorio.
Lanzamiento de moneda: El comando flip_coin lanza una moneda virtual y muestra el resultado.
Fecha de unión al servidor y a Discord: Los comandos joined_server y joined_discord muestran la fecha en que un usuario se unió al servidor y a Discord, respectivamente.
Pruebas: El comando test es un comando de prueba que no hace nada por ahora.
